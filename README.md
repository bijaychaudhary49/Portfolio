# Bijay Kumar Chaudhary — Portfolio

A modern portfolio website showcasing projects, experience, and design skills.

## About

I am a Frontend Developer and UI/UX & Graphic Designer with experience building responsive, accessible web applications using React, Tailwind CSS, and modern design tools.

## Features

- Personal portfolio layout with project showcase
- Contact and social links for GitHub and LinkedIn
- Responsive UI built with React and Tailwind CSS
- Clean code structure using Vite and React Router
- Visual branding with custom icons and imagery

```bash

npm run dev
```

Then open the local Vite URL shown in the terminal.
